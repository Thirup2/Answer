# 练习题1.1
## A.
由题可知：
$\alpha ={1500\over 2500}=0.6$；
$k={150\over 100}=1.5$；
所以：
$S={1\over {(1-\alpha)+{\alpha \over k}}}={1\over {(1-0.6)+{0.6\over 1.5}}}=1.25\times$

## B.
首先假设提升后的速度是：$v$
那么：
$\alpha = {1500\over 2500}=0.6$；
$k={v\over 100}$；
所以：
$S={1\over {(1-\alpha)+{\alpha \over k}}}={1\over {(1-0.6)+{0.6\over{v\over 100}}}}={1\over {0.4+{60\over v}}}=1.67\times$
